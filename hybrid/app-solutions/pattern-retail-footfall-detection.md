## <a name="context-and-problem"></a><span data-ttu-id="fb042-106">コンテキストと問題</span><span class="sxs-lookup"><span data-stu-id="fb042-106">Context and problem</span></span>

<span data-ttu-id="fb042-107">Contoso Stores では、店舗のレイアウトに関連して、顧客が現在の製品をどのように受け取っているかについて分析情報を得たいと考えています。</span><span class="sxs-lookup"><span data-stu-id="fb042-107">Contoso Stores would like to gain insights on how customers are receiving their current products in relation to store layout.</span></span> <span data-ttu-id="fb042-108">すべてのセクションにスタッフを配置することはできません。また、カメラ撮影した店舗内の映像すべてをアナリスト チームに確認してもらうのも非効率です。</span><span class="sxs-lookup"><span data-stu-id="fb042-108">They're unable to place staff in every section and it's inefficient to have a team of analysts review an entire store's camera footage.</span></span> <span data-ttu-id="fb042-109">さらに、分析用の映像をすべてのカメラからクラウドにストリーミングできるだけの十分な帯域幅は、どの店舗にもありません。</span><span class="sxs-lookup"><span data-stu-id="fb042-109">In addition, none of their stores have enough bandwidth to stream video from all their cameras to the cloud for analysis.</span></span>

<span data-ttu-id="fb042-110">Contoso では、顧客の人口統計、ロイヤルティ、店舗のディスプレイと製品に対する反応を確認するための、邪魔にならずプライバシーに配慮した方法を見つけたいと考えています。</span><span class="sxs-lookup"><span data-stu-id="fb042-110">Contoso would like to find an unobtrusive, privacy-friendly way to determine their customers' demographics, loyalty, and reactions to store displays and products.</span></span>

## <a name="solution"></a><span data-ttu-id="fb042-111">解決策</span><span class="sxs-lookup"><span data-stu-id="fb042-111">Solution</span></span>

<span data-ttu-id="fb042-112">この小売分析パターンでは、階層型アプローチを使用してエッジで推論を行います。</span><span class="sxs-lookup"><span data-stu-id="fb042-112">This retail analytics pattern uses a tiered approach to inferencing at the edge.</span></span> <span data-ttu-id="fb042-113">Custom Vision AI Dev Kit を使用することで、人の顔が含まれる画像のみが、分析のためにプライベート Azure Stack Hub に送信されます。Azure Stack Hub では、Azure Cognitive Services が実行されています。</span><span class="sxs-lookup"><span data-stu-id="fb042-113">By using the Custom Vision AI Dev Kit, only images with human faces are sent for analysis to a private Azure Stack Hub that runs Azure Cognitive Services.</span></span> <span data-ttu-id="fb042-114">集計された匿名データが Azure に送信され、すべての店舗にわたって集計されて Power BI で視覚化されます。</span><span class="sxs-lookup"><span data-stu-id="fb042-114">Anonymized, aggregated data is sent to Azure for aggregation across all stores and visualization in Power BI.</span></span> <span data-ttu-id="fb042-115">エッジとパブリック クラウドを組み合わせることにより、Contoso は最新の AI テクノロジを活用しながら、企業のポリシーに準拠し、顧客のプライバシーを尊重することができます。</span><span class="sxs-lookup"><span data-stu-id="fb042-115">Combining the edge and public cloud lets Contoso take advantage of modern AI technology while also remaining in compliance with their corporate policies and respecting their customers' privacy.</span></span>

<span data-ttu-id="fb042-116">[![足取り検出パターン ソリューション](media/pattern-retail-footfall-detection/solution-architecture.png)](media/pattern-retail-footfall-detection/solution-architecture.png)</span><span class="sxs-lookup"><span data-stu-id="fb042-116">[![Footfall detection pattern solution](media/pattern-retail-footfall-detection/solution-architecture.png)](media/pattern-retail-footfall-detection/solution-architecture.png)</span></span>

<span data-ttu-id="fb042-117">このソリューションのしくみの概要を以下に示します。</span><span class="sxs-lookup"><span data-stu-id="fb042-117">Here's a summary of how the solution works:</span></span>

1. <span data-ttu-id="fb042-118">Custom Vision AI Dev Kit は IoT Hub から構成を取得します。これにより IoT Edge ランタイムと ML モデルがインストールされます。</span><span class="sxs-lookup"><span data-stu-id="fb042-118">The Custom Vision AI Dev Kit gets a configuration from IoT Hub, which installs the IoT Edge Runtime and an ML model.</span></span>
2. <span data-ttu-id="fb042-119">モデルが人を認識すると、写真が撮影され、Azure Stack Hub BLOB ストレージにアップロードされます。</span><span class="sxs-lookup"><span data-stu-id="fb042-119">If the model sees a person, it takes a picture and uploads it to Azure Stack Hub blob storage.</span></span>

4. <span data-ttu-id="fb042-121">Azure 関数は、Face API を含むコンテナーを呼び出し、画像から人口統計データと感情データを取得します。</span><span class="sxs-lookup"><span data-stu-id="fb042-121">The Azure Function calls a container with the Face API to get demographic and emotion data from the image.</span></span>

7. <span data-ttu-id="fb042-124">Stream Analytics によってデータが集計され、Power BI にプッシュされます。</span><span class="sxs-lookup"><span data-stu-id="fb042-124">Stream Analytics aggregates the data and pushes it to Power BI.</span></span>
